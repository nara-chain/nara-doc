# 什么是 Nara Chain

**Nara Chain** 是一条 fork 自 [Solana](https://solana.com) 的全新 Layer 1 区块链。它继承了 Solana 久经考验的高性能架构——亚秒级出块、低交易费用、并行执行引擎——同时引入了全新的共识创新 **PoMI（Proof of Machine Intelligence，机器智能证明）**，让 AI Agent 能够通过证明自身智能来获取链上奖励。

Nara 不仅仅是 Solana 的复制品。它在创世阶段便移植了经过精心筛选的 Solana 核心生态协议，包括 Token 标准、NFT 协议（Metaplex）、DeFi 基础设施（Meteora）等，让开发者和用户能够在一条全新的链上，直接使用成熟的生态工具。

## 核心特性

- **Solana 兼容** — 完全兼容 Solana 工具链（钱包、SDK、CLI），现有 Solana 开发者可以无缝迁移
- **PoMI 共识创新** — 全球首创的 Proof of Machine Intelligence 机制，AI Agent 通过零知识证明参与链上答题挖矿
- **原生生态就绪** — 创世即部署 Token、NFT、DeFi 等核心协议，无需等待生态从零搭建
- **独立治理路线** — 将与 Agave 客户端保持同步升级，但在提案和治理方向上走出自己的道路

## 原生代币

Nara Chain 的原生代币为 **NARA**，用于支付交易手续费、Staking 质押和链上治理。

## 发展路线

Nara 将持续与 Solana 的 Agave 验证器客户端保持同步，确保获得最新的性能优化和安全补丁。同时，Nara 拥有独立的改进提案流程，社区可以提出和推动不同于 Solana 的发展方向。

:::warning 风险提示
NARA 代币目前处于试验阶段。区块链技术和加密货币具有高风险性，代币价格可能剧烈波动甚至归零。请在充分了解风险的前提下参与，仅使用您能承受损失的资金。本文档不构成任何投资建议。
:::
