# 运行验证者节点

验证者是 Nara 网络的基石。通过运行验证者节点，您可以帮助保护链的安全、处理交易并获得奖励。

## 开始

官方验证者软件和完整的设置说明在 GitHub 仓库中维护：

**[https://github.com/nara-chain/nara-validator](https://github.com/nara-chain/nara-validator)**

请参阅仓库 README 了解：

- **系统要求** — 硬件和操作系统建议
- **安装** — 从源码构建或使用预编译二进制文件
- **配置** — 设置验证者身份和投票账户
- **运行** — 启动验证者并加入集群
- **监控** — 检查验证者健康状况和性能
- **升级** — 保持节点更新

## 网络端点

确保您的验证者配置连接到正确的网络：

| 网络 | RPC 端点 |
|---|---|
| 主网 | `https://mainnet-api.nara.build/` |
| 开发网 | `https://devnet-api.nara.build/` |

详见 [网络信息](/docs/getting-started/network-info)。
