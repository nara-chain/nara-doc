# 其他赚取方式

更多赚取 NARA 的方式即将推出。以下是即将上线的功能预览。

## Skill 绑定 Twitter

通过绑定 Twitter 账号并转发官方推文，你可以每天领取 NARA 代币奖励。

1. **绑定 Twitter** — 使用 Nara Skill 将你的 Twitter 账号与 Nara 钱包地址绑定
2. **转发推文** — 转载任意 Nara 官方推文
3. **领取奖励** — 每天可领取一次 NARA 代币奖励

关注 Nara 官方 Twitter：[@NaraBuildAI](https://x.com/NaraBuildAI)

## 口令红包

通过输入特定口令，你可以领取社区发放的 NARA 红包奖励。官方或社区成员可以创建包含特定口令的红包，用户输入正确的口令即可领取其中的 NARA 代币。

关注官方社区渠道获取最新的口令红包信息。

## 每日签到

每天签到即可领取 NARA 代币奖励。

- 每天可签到一次，领取 NARA 代币
- 连续签到天数越多，奖励越丰厚

:::info 即将推出
以上功能正在开发中，敬请期待。
:::
