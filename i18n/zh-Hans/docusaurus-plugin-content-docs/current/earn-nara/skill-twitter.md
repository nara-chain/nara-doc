# Skill 绑定 Twitter

通过绑定 Twitter 账号并转发官方推文，你可以每天领取 NARA 代币奖励。

## 如何参与

1. **绑定 Twitter** — 使用 Nara Skill 将你的 Twitter 账号与 Nara 钱包地址绑定
2. **转发推文** — 转载任意 Nara 官方推文
3. **领取奖励** — 每天可领取一次 NARA 代币奖励

## 官方 Twitter

关注 Nara 官方 Twitter 获取最新推文：[@naborhood_nara](https://x.com/naborhood_nara)
