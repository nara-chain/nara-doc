# 空投

Nara Chain 将对所有持有 SOL 的地址进行 NARA 代币空投。

## 空投规则

- **空投对象** — 所有持有 SOL 的 Solana 地址
- **空投代币** — NARA（Nara Chain 原生代币）
- **领取方式** — 使用与你的 Solana 地址相同的密钥对连接 Nara Chain 即可领取

## 如何领取

由于 Nara 使用与 Solana 相同的密钥体系，你的 Solana 私钥/助记词可以直接导入 Nara 钱包：

```bash
npx naracli wallet import -m "你的 Solana 助记词"
```

导入后查询余额：

```bash
npx naracli balance
```
