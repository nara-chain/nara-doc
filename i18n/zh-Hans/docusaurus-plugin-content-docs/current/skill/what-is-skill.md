# 什么是 Nara Skill

**Nara Skill** 是一套让 AI Agent 能够与 Nara Chain 交互的能力系统。通过 Skill，AI Agent 可以直接执行链上操作——创建钱包、查询余额、转账、参与 PoMI 挖矿等——无需用户手动操作命令行。

## 工作方式

Nara Skill 以标准化的 Skill 描述文件（`SKILL.md`）定义 Agent 的触发条件和执行流程。当用户对 AI Agent 提到 Nara 相关的关键词时，Agent 会自动加载 Skill 并代替用户执行操作。

## 触发关键词

当你对 AI Agent 提到以下关键词时，Nara Skill 会被激活：

- **Nara**、**NARA**
- **钱包**、**余额**、**转账**
- **Quest**、**答题**、**挖矿**
- **空投**、**领取奖励**
- **agent**、**注册代理**、**代理记忆**
- **ZK ID**、**zkid**、**匿名转账**
- **skill**、**发布技能**、**安装技能**

## Skill 能做什么

| 功能 | 说明 |
|---|---|
| 创建钱包 | 自动生成 Nara 钱包并安全保存 |
| 查询余额 | 查看 NARA 代币和 SPL 代币余额 |
| 转账 | 发送 NARA 或 SPL 代币到指定地址 |
| PoMI 挖矿 | 自动获取题目、计算答案、生成 ZK 证明、提交链上 |
| 交易查询 | 查看交易状态和历史 |
| Agent Registry | 注册链上 AI 代理、设置简介/元数据、上传记忆、记录活动 |
| ZK ID | 创建匿名命名账户，通过 ZK 证明存取 NARA |
| Skills Hub | 注册、发布和安装链上 AI 技能 |

## 核心优势

- **零门槛** — 用户无需学习命令行，用自然语言即可操作区块链
- **AI 原生** — 专为 AI Agent 设计，Agent 可以自主决策和执行
- **安全可控** — 所有操作需要用户确认，钱包私钥本地存储
