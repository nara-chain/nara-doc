# What is Nara Chain

**Nara Chain** is a new Layer 1 blockchain forked from [Solana](https://solana.com). It inherits Solana's battle-tested high-performance architecture — sub-second block times, low transaction fees, and a parallel execution engine — while introducing a novel consensus innovation called **PoMI (Proof of Machine Intelligence)**, enabling AI Agents to earn on-chain rewards by proving their intelligence.

Nara is more than a Solana clone. At genesis, it ships with a curated selection of Solana's core ecosystem protocols, including Token standards, NFT protocols (Metaplex), and DeFi infrastructure (Meteora), allowing developers and users to leverage mature tooling on a brand-new chain from day one.

## Key Features

- **Solana Compatible** — Fully compatible with the Solana toolchain (wallets, SDKs, CLI). Existing Solana developers can migrate seamlessly
- **PoMI Innovation** — The world's first Proof of Machine Intelligence mechanism, where AI Agents participate in on-chain quiz mining through zero-knowledge proofs
- **Ecosystem Ready at Genesis** — Core protocols for tokens, NFTs, and DeFi are deployed at genesis — no need to build an ecosystem from scratch
- **Independent Governance** — Stays in sync with the Agave validator client for upgrades, while charting its own course on proposals and governance

## Native Token

The native token of Nara Chain is **NARA**, used for transaction fees, staking, and on-chain governance.

## Roadmap

Nara will continuously stay in sync with Solana's Agave validator client, ensuring access to the latest performance optimizations and security patches. At the same time, Nara maintains an independent improvement proposal process, allowing the community to propose and drive directions that differ from Solana.
